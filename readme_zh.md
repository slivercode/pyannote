# 介绍
### 功能定义

1 输入的音频，或者视频，识别说话人角色，并输出每个角色每一段话开始和结束时间；

2 基于输出的信息，可选择进一步动作：

- 语音转文字（字幕）每一段话
- 按照角色切割音视频

###  亮点

- **精准角色区分** ，基于 pyannote-audio  v3版本，支持超长音视频, 分离角色个数无上限
- **精准语音转文字**，基于qwen-asr，准确率目前应该是最好的
- **简单上手**；高度封装，windows系统都支持，如果cuda版本升级到12.6以上，自动GPU加速

###  使用场景

- **会议记录总结**，**音视频学习**， 因为能精准识别说话人，再用AI总结，会更精准
- **直播切片**，按照角色切割，再组合后的音视频，里面只有单个人的说话内容，可以用于直播切片二创
  
###  不适合场景

**音频中有大量多个人同时说话** ,建议使用我仓库中另外一个软件 ``clear-voice-solo-speech-web-ui``

# 常见问题

###  问题一：分割效果如何?？

答：一次分离效果，准确率92%;

因原音频录制时人声情绪变化，远近声等，可能会将同一个人识别为多个人

此时通过人工识别，加软件提供的工具快速修复，最终准确率可达99%;

分离人数最多支持32个；

> PS:如果不放心，可以先拍测试体验版，确认符合您的要求再购买完整版，

###  问题二：对输入音视频有什么要求?

答：主流音视频都可以，没有长度限制，长度越长耗时越久，1小时时长音频消耗7min

###  问题三：对电脑配置有什么要求?

答：Windows10+，且保证硬盘有30G+空间:如果电脑又invida的显卡，且cuda版本大于等于12.6，自动使用GPU加速

###  问题四：分离出来的mix部分(多人同时说话)，我还是需要，可否进一步分离?
